# Escape Tsunami For Brainrots - Enhanced Version

A tycoon game where players collect Brainrots, upgrade them, and survive various disasters like tsunamis, earthquakes, and volcanoes.

## Features
- Collect Brainrots around the map
- Place and upgrade Brainrots in your base for money generation
- Upgrade your base for more slots
- Survive disasters: Tsunami, Earthquake, Volcano
- Offline earnings
- Leaderboards
- Multiplayer

## How to Publish
1. Open Roblox Studio
2. Create a new Baseplate
3. Import the scripts:
   - Place ServerScripts/*.lua into ServerScriptService
   - Place ClientScripts/*.lua into StarterPlayerScripts
   - Place SharedScripts/*.lua into ReplicatedStorage
4. Create models for Brainrot, Base, SafeZone, etc.
5. Add UI elements as needed
6. Test and publish

## Improvements over Original
- Added multiple disasters
- Offline earnings calculation
- Leaderboards
- Better UI
- Random disaster intervals
- Penalty for not escaping disasters
